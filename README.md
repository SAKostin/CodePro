# ProBiology

ProBiology is a console-based electronic biology school implemented in C++.  
The program allows users to register, log in, read a biology article, and take quizzes with automatic scoring.

---

## Features

- User registration and login with duplicate username check.
- Read a short biology article before taking the quiz.
- Two quiz versions with 15 questions each (plants, humans, animals).
- Points calculation and automatic grade assignment.
- Leaderboard displaying users ranked by score.

---

## Menu Structure

1. **Register** — create a new user account.
2. **Login** — log into the system.
3. **Read Article** — read the biology material.
4. **Test** — take the quiz (only after registration and login).
5. **Leaderboard** — view the ranking of all users.
6. **Exit** — exit the program.

---

## Technologies

- C++ (console application)
- Standard libraries: `<iostream>`, `<string>`, `<cstdlib>`, `<ctime>`, `<windows.h>` (for Cyrillic support if needed)
